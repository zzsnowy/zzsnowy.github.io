属性与局部变量不同点

属性是去掉get、set之后的名字，成员变量是定义在外面的变量。

  1. 属性在外面 局部变量在里面

  2. 属性可以用权限修饰符 局部变量不可

  3. 属性根据类型有默认初始化值，而局部变量没有

  4. 在内存中加载的位置，属性加载到堆空间（非static），而局部变量加载到栈空间
